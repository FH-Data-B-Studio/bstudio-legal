# Política de Privacidade

**Última atualização:** 06 de Fevereiro de 2026

## 1. Visão Geral
O aplicativo **B Studio Data Pipeline** é uma ferramenta de uso interno desenvolvida pela FH Data Analysis para fins de inteligência de negócios e análise de dados.

## 2. Dados Coletados
Utilizamos as APIs da Meta (Facebook Graph API e Marketing API) para coletar dados estritamente relacionados à performance de negócios das contas conectadas. Os dados incluem:

* **Métricas de Engajamento:** Alcance, impressões, curtidas, comentários, compartilhamentos e visualizações de vídeo.
* **Dados Demográficos Agregados:** Informações anônimas sobre idade, gênero e localização do público (sem identificação pessoal).
* **Performance de Anúncios:** Custos, cliques (CTR), conversões e resultados de campanhas pagas.

## 3. Uso dos Dados
Os dados coletados são utilizados exclusivamente para:
1.  Geração de dashboards de performance internos;
2.  Cálculo de ROI (Retorno sobre Investimento);
3.  Histórico de crescimento de perfil.

**Nenhum dado é vendido, alugado ou compartilhado com terceiros externos.**

## 4. Retenção e Exclusão de Dados
Os dados são armazenados em nosso Data Warehouse seguro (BigQuery) enquanto o contrato de prestação de serviços estiver ativo.

Caso deseje revogar o acesso ou solicitar a exclusão dos dados coletados, você pode remover o aplicativo através das configurações do Facebook ou entrar em contato conosco.
