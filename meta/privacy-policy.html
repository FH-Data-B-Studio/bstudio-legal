Política de Privacidade

Última atualização: 06 de Fevereiro de 2026

1. Visão Geral

O aplicativo B Studio Data Pipeline é uma ferramenta de uso interno desenvolvida para análise de dados e inteligência de negócios.

2. Dados Coletados

Utilizamos as APIs da Meta (Facebook e Instagram) para coletar dados estritamente relacionados à performance de negócios, incluindo:

  • Métricas de engajamento (likes, comentários, compartilhamentos);
  • Dados demográficos agregados (sem identificação pessoal);
  • Dados de performance de anúncios (Marketing API).

3. Uso dos Dados

Os dados são utilizados exclusivamente para geração de relatórios internos e dashboards de performance. Nenhum dado é vendido ou compartilhado com terceiros externos.

4. Contato

Para dúvidas ou solicitações de exclusão de dados, entre em contato: suporte@fhdataanalysis.com
